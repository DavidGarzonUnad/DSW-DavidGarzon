Propósito del sitio

A partir de contenidos multimedia y explicaciones visuales, se quiere mostrar estas dos formas de maquetar sitios web, a través del uso de Flexbox o CSS Grid se puede realizar de una manera mucho más facil y poderosa la organizacion de los elementos html a través de propiedades CSS, con las cuales las páginas tendrán mayor flexibilidad en varios tipos de pantalla, y se lograrán realizar diseños para todos los gustos de acuerdo a la creatividad que tengamos.

Flexbox

En flexbox tenemos un contenedor y uno o varios hijos. El contenedor es el elemento al que le vamos a indicar que se ponga en disposición flex. Esto hará que todos los elementos hijo inmediatos se coloquen a lo largo de un eje principal.

En cuanto a compatibilidad, Flexbox es actualmente compatible con los navegadores web más importantes y diferentes versiones de los mismos.

El W3C (World Wide Web Consortium) define a esta estructura como un "flujo flexible” de elementos en dirección (arriba/abajo/izquierda/derecha) y tamaño (anchura/altura) según los ejes principal (horizontal) y transversal (vertical).

CSS Grid

Trabaja con el diseño de cuadrícula aplicando reglas CSS tanto a un elemento principal (que se convierte en el contenedor de cuadrícula) como a los elementos secundarios de ese elemento (que se convierten en elementos de cuadrícula).
